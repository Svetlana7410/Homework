# Подсказка по GIT

Создание репозитория:
```sh
git init
```

Проверка статуса
```sh
git status
```

Добавим (сохраненим) изменения в коммит
```sh
git add filename
```

Передача сообщения о совершённых изменениях (комментарий)
```sh
git commit -m "Message"
```

git log
git log --oneline
git checkout
